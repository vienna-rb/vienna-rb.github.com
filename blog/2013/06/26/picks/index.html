Picks / What the vienna.rb Team Thinks Is Worth Sharing This Week

26/6 Picks!!

In a little series on this website we’ll entertain YOU with our picks - or: what we think is worth sharing - every week. Books, articles, gems, funsies… you’re in for an eclectic mix! So, here’s for the tenth edition:

Andy:
  • Performance Optimization With Pry-debugger - A nice introduction to the Pry-debugger
  • The Boy Who Loved Math - The Improbable Life of Paul Erdös. A cute book for kids.
  • Where everyday is Caturday - Need a nice kitten as a placeholder? Or just a random kitten for your chatbot?
Tony: (with support from @fadr)
  • Rails 4.0 - Final version released!
  • Vertical Centering with CSS - Huge guide, nice explanation of the css methods used.
  • Responsive Patterns - Interesting approach for designs that change with media sizes.
Floor:
  • GitHub Similar Repositories - Great Google Chrome add-on, that helps you discover & upvote similar git repos on GitHub.
  • Teach Like You Don’t Know - A ‘serial beginner’ on how teachers can’t relate to their students because they know too much. Good read!
  • deploy via Git - Straight-forward walkthrough deploying via Git.
