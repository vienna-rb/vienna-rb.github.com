Picks / What the vienna.rb Team Thinks Is Worth Sharing This Week

02/10 Picks!!

In a series on this website we’ll entertain YOU with our picks - or: what we think is worth sharing - every week. Books, articles, gems, fun stuff… you’re in for an eclectic mix! So, here’s for the twenty-fourth edition:

Andy:
  • shellcheck - automatically detects problems with sh/bash scripts and commands.
  • joblint - tests tech job specs for issues with sexism, culture, expectations, and recruiter fails.
  • Secret Token - Ruby on Rails Default Token Database detects if a signing token is known somewhere in the Internet, or not.
Floor:
  • sandi_meter - Introducing: a gem which automates scanning your code versus Sandi Metz' rules.
  • GoogleVis - Frontwise sure knows how to visualize the Takedown Requests dataset from Google!
  • RubyConf 2010 - The Road from Ruby Hacker to Entrepreneur (by GitHub’s Tom Preston-Werner).
