Picks / What the vienna.rb Team Thinks Is Worth Sharing This Week

10/12 Picks!

In a series on this website we’ll entertain YOU with our picks - or: what we think is worth sharing - every week. Books, articles, gems, fun stuff… you’re in for an eclectic mix! So, here’s for the one-hundred-eightieth edition:

Laura
  • Coraline Ada Ehmke on CodeNewbie - A really insightful, intense and interesting episode of CodeNewbie about Social Justice.
  • linter-alex - An atom plugin to make writing gender-neutral and less inconsiderate.
  • How to find your first Open Source bug to fix - a very beginner-friendly, short guide to contributing to Open Source
Markus
  • How Dropbox securely stores your passwords - Dropbox has an interesting approach on how they store their passwords.
  • Security Design: Stop Trying to Fix the User - Bruce Schneier argues that user security education is often counterproductive.
  • Nanolaw with Daughter - A sci-fi short story about a possible lawsuit-filled future.
Max
  • How it feels to learn JavaScript in 2016 - Hilarious and somehow sad story on the current state of frontend development.
  • whirly - Terminal spinners for your Ruby CLI.
  • typora - Super neat, minimalistic and promising looking Markdown editor (free during beta).
Gerald
  • Vienna.rb Goes Minima - Vienna.rb Site Redesign using “Plain Vanilla” Jekyll w/ Gem-Packaged Minima Theme.
  • Jekyll 3.3 - What’s News? - Gem-Packaged Themes Now w/ Assets, New relative_url, absolute_url filters, and much more.
  • Qk/Quik Gem-Packaged Jekyll Theme Starter Scaffold - a new scaffold for gem-packaged jekyll themes that mirrors/copies the jekyll new-theme command.
  • Dr Jekyll’s Top 22 Themes - top 22 jekyll themes ranked by github stars; and the winner is…
