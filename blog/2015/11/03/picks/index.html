11/03 Picks!

In a series on this website we’ll entertain YOU with our picks - or: what we think is worth sharing - every week. Books, articles, gems, fun stuff… you’re in for an eclectic mix! So, here’s for the one-hundred-thirty-first edition:

Laura:
  • Rails Rumble 2015 - Registration for the Rails Rumble 2015 opened a couple of days ago. Sign up now!
  • How to solve coding anti-patterns for Ruby rookies - A look at the typical patterns that can be found in beginners' code
  • Codepoems - An experimental IDE for Processing
Floor:
  • CSSgram - A library for editing your images with Instagram-like filters directly in CSS. Interesting? Yes. Useful? Maybe.
  • GitHub on-demand training - Start learning and teaching Git and GitHub today with their self-paced training module.
  • There are RubyIssues. Again. - Contribute to this month’s curated open source construction areas in Ruby.
  • The Pretentious Gem - Very much not recommended but still funny: write your code first and then generate tests later.
Markus:
  • Your App Is Good And You Should Feel Good - Design process, emotions, criticism, and psychology. And how all of that fits together.
  • “Why should I care about customer support?” - Caring about your users as a developer means caring about customer support.
  • The Dynamic Def - What happens when you define a method inside another method in Ruby? Jamis Buck has the answer, and a surprisingly useful use case.
Gerald:
  • Jekyll 3.0 Release - What’s News - Congrats to Parker Moore and team; Jekyll 3.0 - static site builder - now includes a new Liquid template profiler, unified posts and collections, hooks and much more
  • Video - Jekyll 3.0 Talk, Q&A and Live Release by Parker Moore - YouTube video (20 mins) of last week’s Jekyll 3.0 talk by Parker Moore in San Francisco, California
  • Jekyll 3 Assets - Congrats to Jordon Bedwell for updating the asset pipeline for use with the latest and greatest Jekyll version
  • Vienna.html - Vienna Static Site User Group; 1st Meetup on Tue Nov/24th @ sektor5 (Siebenbrunnengasse 44, Wien 5) - Join Us - Free - No Database Required ;-)
